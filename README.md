# Lo Simple

Sitio web estático para presentar y vender el journal Lo Simple.

## Páginas

- `index.html`: página principal.
- `compra.html`: página de compra del journal.

## Archivos necesarios

Para subirlo a GitHub Pages, el repositorio debe incluir:

- `index.html`
- `compra.html`
- `styles.css`
- `script.js`
- `assets/`
- `.nojekyll`

## Publicación en GitHub Pages

En GitHub:

1. Subir todos los archivos y la carpeta `assets`.
2. Entrar a `Settings`.
3. Entrar a `Pages`.
4. En `Source`, elegir `Deploy from a branch`.
5. En `Branch`, elegir `main` y `/root`.
6. Guardar.

GitHub generará un enlace público para compartir con el cliente.
